A Claude Cowork skill for the full job application cycle.

Candidate profile, ATS-safe resumes, timing-aware job search, referral checks, interview prep, and offer negotiation — wired into one workflow. Loads reference files on demand. No account or database required.

Install GitHub
What it does

End-to-end, not feature-by-feature.

One workflow spans intake through offer. You can enter at any phase — Phase 2 to review a resume, Phase 10 to prep for an interview tomorrow — without running Phase 1 first.

Setup · 0–2
0
Reality check. 3 questions → assigns a tier (light / standard / deep).
1
Candidate intake. Builds a profile from your resume; flags LinkedIn mismatches.
2
Resume assessment. ATS compliance, red flags, 1-page compression.
Search & apply · 3–7
3
Job search. Live search via MCP; flags postings older than 2 weeks.
4
Company research & referral check. Checks your network before cold-applying.
5
Per-JD resume. Keyword matrix, gap analysis, targeted rewrite.
6
Cover material. Decides format (formal / short note / none) and drafts it.
7
Submit & follow-up. One follow-up scheduled day 5–7. Never more.
Track · 8–9
8
Tracking. Flat applications.md by default. Notion / Sheets optional.
9
Response-rate diagnostics. Triggers at 10+ apps with < 2 replies; surfaces root cause.
Close · 10–11
10
Interview prep. STAR stories, question bank, thank-you email.
11
Offer & negotiation. Package evaluation, counter scripts, anchoring.
Three tiers

Structural overhead scales to your capacity.

Phase 0 picks one based on available time, clarity of target, and urgency. You can switch tiers between phases.

TierForSetupCovers
lightCasual / exploratoryMinimalSkips SSOT, dossier, most covers
standardActive searchFull flowDefault — everything as documented
deepCareer pivot · < 4wk runwayProactive SSOTDossier per Medium+ match
Install

Two install paths, both under two minutes.

1

Download the latest .skill

From GitHub Releases, attached to the latest tag.

https://github.com/YizhuangLin/job-application/releases/latest
2

Install in Cowork

Claude Cowork → Settings → Skills → Install from file.

3

Trigger in a new chat

Use any of the phrases below.

1

Clone into your skills folder

cd ~/.claude/skills
git clone https://github.com/YizhuangLin/job-application.git
2

Verify structure

Folder should contain SKILL.md, CHANGELOG.md, and references/.

3

Restart Cowork

Cowork picks up new skills on next session start.
